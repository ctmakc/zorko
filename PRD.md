## 1. Проблема

Владелец кофейни, магазина, ресторана или салона открывает утро с несколькими вопросами:
- Что происходит прямо сейчас?
- Что меня ждёт на этой неделе?
- Что я пропустил вчера?

Ответы разбросаны по 5–8 инструментам: кассовая система, Google Reviews, мессенджер с командой, склад, календарь поставщиков, прогноз погоды, Афиша/2ГИС. Владелец тратит 1–2 часа в день на агрегацию, и всё равно что-то пропускает — а это деньги.

**ЗОРКО** читает все эти источники и каждое утро (и в реальном времени) шепчет владельцу только то, что реально важно.

---

## 2. Продукт одной фразой

> ЗОРКО — это AI-дашборд, который смотрит на весь ваш малый бизнес и каждый день говорит одно: «Вот что ты мог не заметить, и вот что с этим делать».

## 3. Целевые пользователи

### Primary ICP (v1)
**Владелец / управляющий** точки HoReCa или ритейла:
- Кофейня, кафе, небольшой ресторан
- Цветочный магазин, пекарня, барбершоп
- 1–5 точек, 2–15 сотрудников
- Не технарь, но в телефоне всё время
- Уже пользуется iiko / r_keeper / Poster / CloudKassir

### Secondary ICP (v1.5)
- Управляющие сетями 5–30 точек (дашборд на мультилокацию)
- Франчайзи (получает аналитику от франчайзора)

### Grant-target ICP (параллельно)
- Акселераторы МСБ при банках (Сбер, Тинькофф, Альфа)
- Правительственные программы цифровизации бизнеса
- AWS/GCP Activate — технологические гранты

## 5. Функциональный объём — v1

### 5.1 Модуль «Сегодня»
- **Утренний бриф** — 3–4 предложения plain-language на основе данных. Генерирует LLM.
- **Фокус дня** — одна конкретная action с обоснованием (откуда данные, ожидаемый эффект, дедлайн). Пользователь «принимает» или «откладывает».
- **Линза** — центральный виджет: главная метрика дня + сигналы со всех источников по кругу. Интерактивный.
- **Шёпоты** — лента инсайтов, отсортированных по типу: риски / рост / идеи. Каждый шёпот — конкретный факт с источником.

### 5.2 Модуль «Пульс»
- Реал-тайм KPI: выручка, средний чек, количество чеков, возвраты, рейтинг, NPS, маржа, очередь
- Спарклайны за скользящий период
- «Сердцебиение бизнеса» — нормированная частота транзакций

### 5.3 Модуль «Шёпоты»
- Полный список инсайтов за день / неделю
- Три колонки: Риски, Рост, Идеи
- Dismiss / «В план» для каждого шёпота
- Фильтрация по источнику

### 5.4 Модуль «Прогноз»
- 7-дневный прогноз трафика (почасовые столбики)
- Уровень доверия: 0.81 на 3 дня, 0.62 на 7 дней
- Предсказание строится из: исторических данных + погода + события + принятые фокусы
- Таймлайн событий недели (поставки, смены, мероприятия рядом)

### 5.5 Источники данных (коннекторы)
| Источник | Тип интеграции |
|---|---|
| Касса (iiko, r_keeper, Poster, ЮKassa) | API / webhook |
| Склад | API кассы или отдельный |
| Google Reviews / Яндекс.Карты / 2ГИС | Public API |
| Telegram-чат команды | Bot API |
| Google Calendar / Calendly | OAuth |
| Погода | Open-Meteo (free) |
| KudaGo / Афиша / локальные события | Web scraper / API |

## 6. Технический стек (целевой)

```
Frontend:   Next.js 15 (App Router) + React + Tailwind
            → дизайн уже готов в HTML/CSS/JSX, портируем 1:1

Backend:    Node.js / Bun + Fastify
            → очередь задач: BullMQ (Redis)
            → CRON-сборщики по каждому источнику

AI layer:   Gemini Flash 2.0 (основной — via OAuth cloudcode-pa, $0)
            Fallback: Claude Sonnet 4.6 (Anthropic API)
            → генерация шёпотов, бриф дня, фокус дня

Storage:    Supabase (Postgres + realtime) + Redis (кэш, очереди)
Hosting:    AWS (Activate кредиты) или Fly.io/Railway
```

## 8. Грантовая и токеновая стратегия

### AWS Activate (приоритет 1)
- **Программа:** AWS Activate Founders / Portfolio
- **Что даёт:** $5K–$100K кредитов AWS, поддержка SA, ранний доступ к сервисам
- **Требования:** Стартап до Series A, зарегистрирован, нет дублирующих кредитов
- **Наш угол:** AI-native product, heavy on inference + vector storage + realtime data pipelines
- **Что хостим на AWS:** EC2 для воркеров, RDS (Postgres), ElastiCache (Redis), Lambda для коннекторов, Bedrock (если нужен fallback LLM)
- **Применить:** https://aws.amazon.com/activate/ (форма + pitch deck нужен)

### Google for Startups Cloud Program (приоритет 2)
- **Что даёт:** $200K кредитов GCP за 2 года + Vertex AI credits
- **Наш угол:** Используем Gemini Flash через cloudcode-pa → натуральный аргумент для миграции части нагрузки на GCP
- **Применить:** через Google for Startups program

### Microsoft for Startups Founders Hub (приоритет 3)
- **Что даёт:** $150K кредитов Azure + $2500 OpenAI API credits
- **Хорошо для:** fallback LLM, Azure Functions для коннекторов
- **Бонус:** Быстро одобряют, без инвесторского бэкинга

### AI Token-специфичные программы

| Провайдер | Программа | Что даёт |

## 10. Риски

| Риск | Вероятность | Митигация |
|---|---|---|
| Малый бизнес не платит за SaaS | Средняя | Freemium + wow-момент до paywall |
| API кассовых систем закрыты/платные | Средняя | Начать с ЮKassa (открытый API), iiko партнёрка |
| LLM inference стоит дорого | Низкая | Gemini Flash + кэширование + токеновые гранты |
| Конкурент (Sber Analytics, etc.) | Низкая | Скорость + UX класса А (дизайн уже есть) |
| Churn из-за сложного онбординга | Высокая | Onboarding < 5 мин — жёсткий KPI |
